¿Que es CSS?

CSS (cascade style sheet) es lo que le da apariencia a tu pagina web

¿Que es una referencia por palabra clave?

Las palabras clave son palabras o frases que se utilizan para relacionar los anuncios con los términos que buscan los usuarios. La selección de palabras clave relevantes y de alta calidad para su campaña publicitaria puede ayudarlo a llegar a los clientes que desea, en el momento en que lo desea.

¿Que es una referencia por id?

El atributo id nos permite darle un nombre específico a una parte de un documento HTML definida por una etiqueta HTML, diferenciándola de las demás.

¿Que es una referencia por clase?

El atributo class está definido como una lista de elementos separados por espacio, y uno de esos elementos debe coincidir exactamente con el nombre de clase dado en el selector.
